# Caesar's Resurrection - Remotion Video

Animated video of Julius Caesar rising from death with supernatural energy.

## Features

- 6 cinematic frames with smooth transitions
- Blue supernatural energy effects during awakening
- Golden divine light for resurrection
- Dramatic zoom and glow animations
- Title overlay at the end
- 19 seconds total (570 frames at 30fps)

## Quick Start

```bash
# Install dependencies
npm install

# Preview in browser (interactive)
npm start

# Render to MP4
npm run build
```

## Output

Video will be saved to: `out/caesar-resurrection.mp4`

## Customization

Edit `src/CaesarResurrection.tsx` to adjust:
- Frame durations (currently 3 seconds each)
- Transition timing (currently 1 second crossfade)
- Color effects (blue energy vs golden divine light)
- Zoom intensity
- Title text and styling

## Technical Details

- **Resolution:** 1920x1080 (Full HD)
- **FPS:** 30
- **Duration:** 19 seconds
- **Transitions:** Crossfade with zoom
- **Effects:** Radial gradients, glow, vignette
